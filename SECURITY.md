# Security Policy

Report suspected vulnerabilities through [GitHub private vulnerability reporting](https://github.com/daxro/laget-cli/security/advisories/new).

Do not open a public issue for vulnerabilities, credential exposure, or session-cookie exposure. Never include credentials or session cookies in a report.

The latest version on the default branch is supported. This project uses undocumented laget.se endpoints, so upstream changes may break behavior without notice.
